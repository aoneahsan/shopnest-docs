# Admin Dashboard

The admin dashboard is the store-owner side of ShopNest — a separate, access-controlled area for running the store. From it, administrators manage the catalog, fulfill orders, oversee customers and reviews, publish content, and read analytics.

## What admins can manage

The dashboard brings store operations together:

- **Products** — add, edit, and organize catalog items and their details.
- **Orders** — view and update order statuses through fulfillment.
- **Users** — manage customer accounts and roles.
- **Categories** — structure the catalog into browsable groups.
- **Reviews** — moderate shopper [reviews](/features/reviews).
- **Posts & content** — write [blog](/features/blog) articles and manage site content.
- **Analytics** — read sales and activity metrics.
- **Settings** — configure store-wide options.

## Access control

The admin area is gated behind authentication and admin roles, so only authorized accounts reach it. Shoppers never see the dashboard; it's a distinct surface from the storefront.

## Analytics

The analytics view summarizes store activity — orders, sales, and engagement — so an owner can see how the store is doing at a glance. Charts present the data visually for quick reading.

## Frequently asked questions

### Who can access the admin dashboard?

Only authenticated accounts with an admin role. The storefront and the dashboard are separate, access-controlled surfaces.

### Can admins manage the catalog and orders in the same place?

Yes. Products, orders, users, categories, reviews, content, and analytics are all managed from the one dashboard.

### Is content editing code-free?

Yes. Blog posts and content are edited with a rich-text editor, no code required.
