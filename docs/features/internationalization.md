# Internationalization

Internationalization makes ShopNest usable beyond English. The storefront is localized into six languages, with the interface text translated and a language switcher so shoppers can read the store in the language they prefer.

## Supported languages

ShopNest ships translations for:

- **English (en)**
- **Spanish (es)**
- **French (fr)**
- **German (de)**
- **Arabic (ar)** — with right-to-left layout
- **Chinese (zh)**

The app uses i18next for translation management, and a language switcher in the interface lets shoppers change languages on the fly.

## Right-to-left support

Arabic is a right-to-left language, and ShopNest accounts for that in its layout so the interface reads naturally rather than simply swapping words into a left-to-right frame. This makes the store genuinely usable for RTL readers, not just nominally translated.

## What gets translated

Interface text — navigation, buttons, labels, and standard messages — is translated across the supported languages. Product data entered by the store reflects how the store authored it.

## Frequently asked questions

### How many languages does ShopNest support?

Six: English, Spanish, French, German, Arabic, and Chinese.

### Does ShopNest support right-to-left languages?

Yes. Arabic is supported with right-to-left layout handling.

### How do I change the language?

Use the language switcher in the interface to select your preferred language.
