## About ShopNest

ShopNest is a modern e-commerce platform: a storefront with catalog, search, cart, and checkout, plus accounts, loyalty, reviews, and an admin dashboard. It's built with React 19, TypeScript, Vite, TanStack Router/Query, Radix UI, Tailwind, Zustand, and Capacitor, on Firebase/Firestore. The app is free and shows only first-party promotions for the developer's other apps.
